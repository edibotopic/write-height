Write Height | About

This application converts grayscale images into 3D models that can be viewed and downloaded in different file formats. Simple drawing tools are also provided for creating the images.

Controls

You can rotate (left-mouse or finger swipe), zoom (scroll wheel or finger pinch) and pan (right-mouse or two-finger swipe) around the 3D scene to inspect the model.

Images

A set of example images are included for inspiration. Original images can be created using any decent drawing or image-editing application. A minimal drawing tool is included that can be used to draw and download a grayscale image. In addition, grayscale images can be algorithmically generated, although currently only one algorithm is provided.

Exporting

3D models can be exported as .stl, .obj and .glb files for use in other 3D software or 3D printing. Model topology may need to be cleaned up for your specific application.

Motivation

I made this tool to be a free, easy and portable way to experiment with 3D. View source code on Github.

Implementation

The application is largely written using Vanilla JavaScript and relies on Babylon.js for 3D rendering.
