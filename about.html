Write Height | About

This application converts grayscale images into 3D models that can be viewed and downloaded.

Controls

You can rotate (left-mouse or finger swipe), zoom (scroll wheel or finger pinch) and pan (right-mouse or two-finger swipe) around the 3D scene to inspect the model.

Images

A set of example images are included for inspiration. Original images can be created using any decent drawing or image-editing application and then uploaded.

Exporting

3D models can be exported as .stl files for use in other 3D software or 3D printing. Model topology may need to be cleaned up for your specific application.

Motivation

I made this tool to be a free, easy and portable way to experiment with 3D. View source code on Github.

Implementation

The application is written using Vanilla JavaScript and relies on Babylon.js for 3D rendering.

Plans

There is a memory issue that results in multiple downloads being triggered during model export. I am working (slowly) to resolve this.
