Write Height

Convert any greyscale image to 3D.


This simple tool converts greyscale images into 3D using Babylon.js. You can pan (drag) and zoom (pinch/scroll) to inspect the model. Models can be exported as .stl, .obj and .glb files for your projects. I made this tool to be a free and easy way for beginners to experiment with 3D. View source on Github.
