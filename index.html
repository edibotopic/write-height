Write Height | Convert any greyscale image to 3D




This tool converts greyscale images into 3D using Babylon.js. You can rotate (right-mouse or finger swipe), zoom (scroll wheel or finger pinch) and pan (left-mouse or two-finger swipe) to inspect the model. Uploaded models can be exported as .stl, .obj and .glb files. I made this tool to be a free and easy way for beginners to experiment with 3D. View source code on Github.
